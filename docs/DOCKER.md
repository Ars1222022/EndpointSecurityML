# Docker för EndpointSecurityML

## Kom igång
1. Starta Docker Desktop
2. Kör: .\docker.ps1
3. Välj 1 för att starta
4. Öppna http://localhost:5000

## Kommandon
| Kommando | Funktion |
|----------|----------|
| .\docker.ps1 | Meny |
| docker-compose up --build -d | Starta |
| docker-compose down | Stoppa |
